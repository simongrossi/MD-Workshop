# Obsidian vs MD Workshop

Question fréquente : *"Pourquoi ne pas simplement utiliser Obsidian ?"*

Réponse courte : **tu peux**. MD Workshop n'essaie pas de remplacer Obsidian — il propose une autre philosophie.

## Ce qui est pareil

- Fichiers `.md` locaux
- Wiki-links `[[...]]`
- Backlinks, tags, outline
- Vue graphe

## Ce qui diffère

| | Obsidian | MD Workshop |
|---|---|---|
| Taille du bundle | ~100 Mo | ~15 Mo (Tauri) |
| Plugins | ~1500 plugins | aucun |
| Compte / cloud | Optionnel | aucun |
| Rendu | Moteur propre | `marked` + DOMPurify |
| Philosophie | Extensible | Minimaliste |

## Quand choisir quoi

- Besoin d'un système riche, de plugins, d'un canvas visuel → **Obsidian**
- Besoin d'un éditeur Markdown rapide, qui démarre instantanément et reste petit → **MD Workshop**
