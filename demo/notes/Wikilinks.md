# Les wiki-links

Dans MD Workshop, tu peux faire un lien vers une autre note avec la syntaxe `[[Nom du fichier]]`.

## Exemples

- Lien simple : `[[Accueil]]` → [[Accueil]]
- Lien avec alias : `[[Accueil|Page d'accueil]]` → [[Accueil|Page d'accueil]]
- Lien vers sous-dossier : `[[projets/Idées]]` → [[projets/Idées]]

## Autocomplétion

Tape `[[` dans l'éditeur : une liste des fichiers du workspace apparaît. Tu peux filtrer en tapant quelques lettres.

## Liens non résolus

Si tu écris `[[Une note qui n'existe pas]]`, le lien apparaît en surbrillance rouge dans l'aperçu. Un clic propose de créer le fichier.

Essaie : [[Une note fictive]].
