# À propos

**MD Workshop** est un éditeur Markdown desktop, pensé pour être *léger et direct*. Pas de cloud, pas de compte, pas de bloat : juste tes fichiers `.md` dans un dossier local.

## Ce que fait l'outil

- Édition Markdown avec coloration (CodeMirror 6)
- Aperçu HTML synchrone (mode split)
- Index plein-texte via SQLite (recherche rapide)
- Liens `[[wiki]]`, backlinks, tags, outline
- [[projets/Phase 4 - Graphe|Vue graphe]] du réseau de notes

## Ce que l'outil **ne fait pas**

- Pas de synchronisation cloud (à toi d'utiliser Git, Syncthing, etc.)
- Pas de plugins tiers
- Pas de base de données cachée : tes `.md` restent des `.md`

## Inspiration

Pensé comme un [[notes/Obsidian vs MD Workshop|Obsidian minimaliste]], en gardant uniquement les fonctionnalités qui servent tous les jours.
