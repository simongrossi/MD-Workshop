# Bienvenue dans MD Workshop

Ceci est le **dossier de démonstration**. Il est fait pour te montrer (et te laisser tester) les principales fonctionnalités de l'éditeur, sans avoir à créer tes propres notes.

> N'hésite pas à modifier ces fichiers — c'est une copie locale, tout est à toi.

## Par où commencer ?

- [[À propos]] — ce qu'est MD Workshop (et ce que ça n'est pas)
- [[Raccourcis clavier]] — les combos à connaître
- [[notes/Markdown 101]] — rappel rapide de la syntaxe
- [[notes/Wikilinks]] — comment les liens `[[…]]` fonctionnent
- [[notes/Tags]] — les tags `#inline` et frontmatter
- [[projets/Phase 4 - Graphe]] — exemple de note de projet

## À tester absolument

1. Ouvre la **vue graphe** (`Alt+4`) pour voir comment ces notes sont reliées.
2. Clique sur n'importe quel lien `[[...]]` ci-dessus dans l'aperçu — ça ouvre le fichier.
3. Essaie `Ctrl+P` pour sauter à un fichier en tapant quelques lettres.
4. Ouvre `Ctrl+Shift+P` pour voir toutes les commandes.
5. Coche une case dans [[notes/Markdown 101]] depuis l'aperçu : le fichier source est mis à jour.
