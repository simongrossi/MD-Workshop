# Raccourcis clavier

## Navigation

| Raccourci | Action |
|---|---|
| `Ctrl+P` | Aller à un fichier (quick open) |
| `Ctrl+Shift+P` | Palette de commandes |
| `Ctrl+D` | Note du jour |
| `Ctrl+O` | Ouvrir un dossier |
| `Ctrl+Shift+O` | Changer de bibliothèque |

## Affichage

| Raccourci | Action |
|---|---|
| `Alt+1` | Mode édition seul |
| `Alt+2` | Mode split (éditeur + aperçu) |
| `Alt+3` | Mode aperçu seul |
| `Alt+4` | **Mode graphe** ← nouveau |

## Édition

| Raccourci | Action |
|---|---|
| `Ctrl+S` | Enregistrer |
| `Ctrl+Shift+S` | Enregistrer sous… |
| `Ctrl+N` | Nouveau fichier |
| `Ctrl+W` | Fermer l'onglet |

## Recherche

| Raccourci | Action |
|---|---|
| `Ctrl+Shift+H` | Recherche & remplacement multi-fichiers |

## Export / copie

| Raccourci | Action |
|---|---|
| `Ctrl+Shift+W` | Copier pour WhatsApp |
